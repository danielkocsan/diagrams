# Diagrams
Example application using the following technologies:
  - d3.js
  - Require.js
  - Jasmine
  - Karma
  - Sass / Flexbox
## Installation
1. Clone this repository
2. Execute the following lines in the command line:
```sh
npm install
bower install
npm install http-server -g
npm run server
```
## Testing
1. Execute `npm test`
## Development
1. Execute `npm run watch`
